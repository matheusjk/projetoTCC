ATORES -> USUARIO | SISTEMA | ESP8266

USUARIO -> CADASTRAR NO SISTEMA PARA TER ACESSO AOS DADOS
CRIAR USUARIO
RECUPERAR SENHA DE ACESSO
VIZUALIZAR OS OUTROS USUARIOS, LOCAL, TELEMETRIA, GEOLOCALIZAÇÃO NO SISTEMA E DADOS VINDO DO ESP8266


SISTEMA -> DISPONIBILIZAR OS DADOS VINDOS DO ESP8266
TER CONTROLE DE ACESSO AOS DADOS ATRAVES DA TELA DE LOGIN
MOSTRAR DADOS DE TELEMETRIA EM TEMPO REAL
MOSTRAR O QUANTITATIVO DE MODULOS(ESP8266+SENSORES) SENDO USADOS E QUAL SUA POSIÇAO APROXIMADA(TABELA GEO)

ESP8266 -> DISPONIBILIZAR DADOS DE TELEMETRIA E GEOPOSIÇÃO PARA ALIMENTAR O BANCO DE DADOS
LER INFORMAÇÃO DA TABELA DE CONFIGS ONDE ESTARA AS INFOS DE TEMPO PARA ENVIO DOS DADOS DE TELEMETRIA



REQUISITOS FUNCIONAIS

O SISTEMA DEVERA PERMITIR O CADASTRO DE NOVOS USUARIOS E EXCLUSAO
NO SISTEMA O USUARIO CADASTRARA ONDE O EMBARCADO FICARA ALOCADO
O USUARIO DEVERA INFORMAR NA TABELA DE CONFIGS O TEMPO DE ATUALIZACAO DOS DADOS TANTO DE GEOLOCALIZACAO QUANTO DOS DADOS DE TELEMETRIA(SENSOR)
O SISTEMA DEVARA POSSIBILITAR AO USUARIO A RETIRADA DE RELATORIOS
O USUARIO ATRAVES DE FILTROS PODERA PESQUISAR PELA TEMPERATURA, UMIDADE E MUDANÇA NA QUALIDADE DO AR(SENSOR MQ-135)
O SISTEMA DEVERA EXIBIR GRAFICOS DESSES DADOS


REQUISITOS NAO FUNCIONAIS

SISTEMA DEVERA FUNCIONAR EM QUALQUER BROWSER
SISTEMA DEVERA SER ACESSADO POR QUALQUER S.O
A APLICAÇAO SOMENTE SERA ACESSADA APOS O LOGIN E SENHA
USABILIDADE E NAVEGAÇAO SIMPLES
